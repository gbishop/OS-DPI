# Layout Nodes

All possible nodes in the layout tree, except for Tab Control.

## Details
| Type | Leaf? | Possible Children | 
| ---- | ----- | ----------------- |
| Page | N | Stack, Modal Dialog, Speech, Audio |
| Stack | N | Stack, Grid, Display, Radio, Tab Control, VSD, Button |
| Modal Dialog | N | - |
| Speech | Y | n/a |
| Audio | Y | n/a |
| Grid | Y | n/a |
| Display | Y | n/a |
| Radio | Y | n/a |
| Tab Control | Y | n/a |
| VSD | Y | n/a |
| Button | Y | n/a |
